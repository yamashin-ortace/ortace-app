# ORT ACE LP用アプリ画面スクリーンショット素材

保存日: 2026-05-18

このフォルダはLP制作・モック作成用の元スクリーンショット保管場所です。

## 注意

- このフォルダは `public/` 配下ではないため、Webから直接配信されません。
- 元画像にはユーザー名や実際の学習データが含まれる可能性があります。
- LPで使用する場合は、必要部分をトリミングし、個人名・不要情報を隠した加工版だけを `public/landing/app-screens/` に配置します。

## ファイル一覧

- `01-home-dashboard.JPG`: ホーム画面・ダッシュボード
- `02-ai-coach-question-before.JPG`: AIコーチ 今日のおすすめ・解答前
- `03-ai-coach-question-correct.JPG`: AIコーチ 今日のおすすめ・正解後
- `04-ai-coach-question-incorrect.JPG`: AIコーチ 今日のおすすめ・不正解後
- `05-ai-coach-result-analysis.JPG`: AIコーチ 結果分析
- `06-records-bookmarks.JPG`: 記録・ブックマーク一覧
- `07-records-notes.JPG`: 記録・ノート一覧
- `08-records-history-long.JPG`: 記録・履歴ロングスクリーンショット
- `09-study-menu.JPG`: 学習メニュー

## LPで優先的に使いやすい素材

1. `01-home-dashboard.JPG`
   - 今日やることが見えることを伝えるヒーロー/中盤向き。
2. `05-ai-coach-result-analysis.JPG`
   - 解いた後にAIコーチが次の確認テーマを出す差別化訴求向き。
3. `09-study-menu.JPG`
   - 今日のおすすめ、復習、未着手、弱点リペア、思い込みチェックがまとまって見える。
4. `03-ai-coach-question-correct.JPG` / `04-ai-coach-question-incorrect.JPG`
   - 問題演習と解説が実際に使えることを見せるモック向き。
