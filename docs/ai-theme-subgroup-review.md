# AIコーチ サブグループ候補レビュー

目的: 大きすぎるAIテーマクラスタだけを、追加3問が成立する範囲で細分化する。

## 基本方針

- 親クラスタは残す。
- 追加3問は、まずサブグループ内から選ぶ。
- サブグループ内で3問に足りない場合は、親クラスタから補完する。
- 分けた結果、サブグループが1〜2問だけになるものは原則作らない。
- 画面表示は、可能なら誤答テーマ名またはサブグループ名を優先する。

## 追加メモ

- 14番「眼球運動の法則」は、外眼筋の神経支配を分離済み。残りをさらに分けるかは様子見。
- 26番「眼の発生と正常値」は、眼の発生 / 涙液・涙膜 / 角膜の特性・計測 / 眼球計測・正常値に分離済み。
- 29番「眼振」の QOV は、コントラスト感度と合わせて「コントラスト感度・視覚の質」に分離済み。
