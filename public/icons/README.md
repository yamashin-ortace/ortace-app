# アプリアイコン（ORT ACE）

PWA / ホーム画面追加時に使われる PNG アイコンを配置する場所です。

## 必要なファイル

| ファイル名 | サイズ | 用途 |
|---|---|---|
| `icon-192.png` | 192×192 | Android ホーム画面・PWA |
| `icon-512.png` | 512×512 | スプラッシュ画面・高解像度 |
| `icon-maskable-512.png` | 512×512 | Android 丸型マスク対応 |
| `apple-touch-icon.png` | 180×180 | iOS ホーム画面追加 |

## 今の状態

`public/landing/ort-ace-logo-final.png` の左側アイコンマークを正方形に切り出して生成しています。
ブランドロゴを再調整した場合は、同じファイル名で差し替えます。
