# AIコーチ テーマ分類レビュー

目的: AIコーチ分析と「このテーマを3問だけ確認」のズレを減らすため、AIテーマクラスタを目視レビューする。

## 判断ルール

- 追加3問が成立することを優先する。分けた後のクラスタが3問未満になりそうなら、原則として分けない。
- 違和感が強い組み合わせは分ける。例: 眼振 / 頭位異常・代償頭位。
- 問題数が少ない細テーマは、クラスタ名を広めに保ち、分析文では誤答テーマ名を優先表示する。
- チェック時は「同じクラスタ名で追加3問が出ても自然か」を見る。

## 現状サマリー

- 判定対象: 1,490問
- AIテーマクラスタ定義: 57個
- 使用中クラスタ: 57個
- 3問以上あるクラスタ: 57個
- 3問未満のクラスタ: 0個
- 複数テーマを含むレビュー対象クラスタ: 56個
- 完全一致テーマだけで3問成立するテーマ: 48個
- 完全一致テーマだけでは3問未満のテーマ: 1,195個

つまり、追加3問は「完全一致だけ」で選ぶ設計には向いていません。基本は同じAIテーマクラスタ内で選び、誤答テーマに完全一致する問題があれば強く優先する方針が現実的です。
